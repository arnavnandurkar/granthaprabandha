Granthaprabandha (Sanskrit for 'book management') is, as the name suggests, a library management system. This is ideal for a physical collection of books. I started making it to track my own collection of books, which is quite large, but ended up making this comprehensive system which can also be used by other users like you. Enjoy!

> Features
1. Create multiple catalogs and add books according to shelf
2. Add custom genres, shelf names, and formats
3. Track books you've lent to people
4. Add books by scanning barcodes
5. Automatically adds book cover, title, author from ISBN
6. Share your catalogs with a linkt- MVP ready for launch)
